API Albures MX

Página inicial de la API Albures MX

... es una API JSON gratuita de albures clasificados diferentes tipos.

Ejemplos de uso

Obtener los tipos de albures:

GET https://api-albures-2.vercel.app/api/categorias

Ejemplo obtener un albur por categoria:

GET https://api-albures-2.vercel.app/api/{categoria}

Ejemplo de respuesta:

{"tipo":"comparativo","albur":"No es lo mismo la papaya tapatía, que tápate la papaya Tía."}
