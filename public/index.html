AlburesMX

AlburesMX es una API JSON sencilla y gratuita que permite obtener albures clasificados en diferentes categorias. No es necesario crear una cuenta de usuario ni el uso de una clave API para poder usar esta API.

Las categorías disponibles son:

  • comparativo
  • nombre
  • enunciado
  • piropo
  • pregunta

Ejemplos de uso

- Para obtener los tipos de albures:

GET https://api-albures-2.vercel.app/api/categorias

- Para obtener un albur de una categoria específica:

GET https://api-albures-2.vercel.app/api/{categoria}

- Ejemplo de la respuesta recibida:

{
"categoria":"comparativo",
"albur":"No es lo mismo la papaya tapatía, que tápate la papaya Tía."
}

- Para obtener todos los albures de una categoria específica:

GET https://api-albures-2.vercel.app/api/{categoria}/todos
