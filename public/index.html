Albures MX

Página inicial de la API Albures MX

Albures MX es una API JSON gratuita de albures, clasificados diferentes categorias.

Ejemplos de uso

Obtener los tipos de albures:

GET https://api-albures-2.vercel.app/api/categorias

Ejemplo d solicitud para obtener un albur por categoria:

GET https://api-albures-2.vercel.app/api/{categoria}

Ejemplo de la respuesta recibida:

{
"categoria":"comparativo",
"albur":"No es lo mismo la papaya tapatía, que tápate la papaya Tía."
}

Ejemplo de solicitud para obtener todos los albures de una categoria:

GET https://api-albures-2.vercel.app/api/{categoria}/todos
